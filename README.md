# Curso de Python "De 0 a NLP"
Este repositorio contiene el material que diseñé para mi curso **"De 0 a NLP"**, donde enseño a utilizar diversas librerías de Python para procesamiento de lenguaje natural. El curso cubre desde herramientas básicas como Numpy y Pandas hasta librerías más avanzadas como NLTK y las librerías de Hugging Face. Aquí encontrarán todas las lecciones que impartí, organizadas paso a paso, con código, ejemplos y pequeños ejercicios para que los estudiantes realicen durante la lección.
